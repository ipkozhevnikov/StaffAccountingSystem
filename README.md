# Система учёта сотрудников

## Описание проекта
Данный проект представляет собой консольное приложение на C++, которое объединяет два независимых модуля:

1. **Калькулятор** — выполняет математические операции, а также работает со строками и массивами  
2. **Система учёта сотрудников** — позволяет управлять списком сотрудников и сохранять данные в файл  

При запуске пользователь выбирает нужный режим работы.

---

## Функционал

**Команды:**
- `add` — добавить сотрудника  
- `remove` — удалить сотрудника по индексу  
- `list` — вывести всех сотрудников  
- `searchByName` — поиск по имени  
- `searchByPost` — поиск по должности  
- `update` — загрузить данные из файла  
- `save` — сохранить данные в файл  
- `help` — список команд  
- `exit` — выход  

**Данные сотрудника:**
- Имя  
- Фамилия  
- Возраст  
- Зарплата  
- Должность:
  - `programmer`
  - `manager`
  - `analyst`
